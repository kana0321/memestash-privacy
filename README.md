# memestash-privacy

iOS / Android アプリ **Memestash** のプライバシーポリシー公開用リポジトリ。<br>
GitHub Pages で `index.html` を配信し、そのURLを App Store Connect / Google Play のプライバシーポリシー欄に登録する。

`index.html` はデザインシステム「Nightlime」（`memestash-ios`）を踏襲したスタイル。`.nojekyll` で Jekyll 処理を無効化し、静的HTMLをそのまま配信している。<br>
`index.md` は同じ内容のプレーンテキスト版（履歴・参照用）。

ポリシー本文の原本はアプリ本体リポジトリ `memestash-ios` の `docs/privacy-policy.md`。内容を変更する際はそちらを正として、ここへ反映する。
